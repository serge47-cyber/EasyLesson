# 🎓 АІ-Методист: Цифровий дизайнер адаптивних уроків (10-11 класи)

Унікальна інтерактивна веб-платформа, створена спеціально для українських вчителів та школярів старших класів. Цей інструмент миттєво перетворює сухий академічний текст шкільних підручників на захопливі інтерактивні уроки-квести, адаптовані під сучасне сприйняття інформації.

---

## 🎯 Для кого цей застосунок?

1. **Для вчителів старшої школи (Килер-фіча!):**
   * Можливість за 30 секунд сформувати готовий опорний план-конспект уроку.
   * Конструктор карток, квізів та покрокових рішень з миттєвим експортом у **Microsoft Word (.doc)** та **професійний PDF/Друк format**.
   * Авторські методичні стилізації викладання під будь-який предмет.

2. **Для учнів 10-11 класів:**
   * Спрощене засвоєння складних тем НУШ за допомогою яскравих життєвих метафор та аналогій.
   * Тренування пам'яті через активні флеш-картки.
   * Експрес-діагностика знань за допомогою інтерактивних міні-тестів.
   * Покроковий інтерактивний тьютор для розбору складних формул, задач і життєвих викликів.

---

## 🎭 5 Унікальних ШІ-Стилів Викладання

Методист динамічно перевтілюється в одного з експертів залежно від обраного предмету або персональних побажань вчителя:

*   **🕯️ Політичний / Історичний трилер:** (для Історії та суспільних наук) — перетворює сухі дати на динамічне протистояння інтересів, таємних змов, дипломатичних ігор та геополітичних рішень.
*   **🔬 Науковий детектив та інженерія:** (для Фізики, Хімії, Біології, Алгебри) — фокусується на глибинних законах природи, фізичних парадоксах та інженерних хаках нашого всесвіту.
*   **🎭 Психопрофайлінг та мотиви:** (для Літератури, Мистецтва, Психології) — досліджує внутрішній світ героїв, психотипи персонажів та приховану душевну мотивацію авторів.
*   **🗣️ Лінгвістичний коучинг:** (для Іноземних мов) — спрямований на інтерактивне занурення, розмовну практику та культурні сленгові тонкощі без зазубрювання.
*   **🧩 Інтелектуальний квест та логіка:** (для Універсальних та системних дисциплін) — будує цілісну причинно-наслідкову картину знань крізь логічні ланцюжки.

## 🎨 Як виглядає результат?

Кожен генерований урок складається з 5 зручних модулів:

1.  **🎮 Вступний брифінг (Mission Briefing):** Невелика інтерактивна місія, яка занурює учня в процес дослідження та ставить перед ним практичну задачу.
2.  **⚡ Конспект-Тези:** Ключова вижимка теорії. При кліку на кожну тезу відкривається детальна **асоціативна метафора** (пояснення складного явища за допомогою простих побутових речей).
3.  **📋 Флеш-картки:** Термінологічний словник для швидкого запам'ятовування, який можна перевертати в один клік.
4.  **❓ Бліц-квіз:** Інтерактивні тести з миттєвою візуальною перевіркою відповідей та детальним обґрунтуванням помилок.
5.  **🧩 Розбір задач (Step Solver):** Деталізоване покрокове вирішення практичних вправ із виокремленням базових принципів та проміжними результатами.

## ⚙️ Технічні вимоги та локальний запуск

Додаток побудований на сучасному та швидкому стеку технологій **React 19 + TypeScript + Vite + Express (API proxy) + Tailwind CSS + SDK Gemini (@google/genai)**.

### Передумови
Передбачається, що у вас встановлено **Node.js v18+**.

1. **Встановлення залежностей:**
   ```bash
   npm install
   ```

2. **Налаштування змінних оточення:**
   Створіть файл `.env` у кореневому каталозі та додайте свій ключ API від Google AI Studio:
   ```env
   GEMINI_API_KEY=ваш_секретний_ключ_gemini
   ```

3. **Запуск у режимі розробки (Development):**
   ```bash
   npm run dev
   ```
   Додаток буде доступний за адресою [http://localhost:3000](http://localhost:3000).

4. **Збирання для продуктивного використання (Production build):**
   ```bash
   npm run build
   npm run start
   ```

## 🚀 Особливості розгортання та холодна оптимізація (Render.com)

Додаток адаптований для стабільної роботи на безкоштовному хмарному хостингу **Render.com**:
*   **Індикатор холодної ініціалізації:** Оскільки безкоштовні сервери Render переводяться в сон після 15 хвилин неактивності, додаток має автоматичний пінгувальник на фронтенді, який делікатно пояснює користувачу стан запуску контейнера у перші 30–40 секунд та відображає прогрес-бар.
*   **Локальний PDF парсер:** Оцифрування великих шкільних підручників відбувається безпосередньо у браузері користувача за допомогою оптимізованого модуля `pdf.js` через хмарний CDN, що знімає навантаження з сервера і виключає падіння контейнера за лімітами оперативної пам'яті. У разі обмежень браузера передбачений автоматичний фоллбек на серверний парсинг.
